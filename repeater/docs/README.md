# docs — valmistusohje ja whitepaperit

Repeater-projektin **spesifikaatio ja suunnitteludokumentit**. Tämä on projektin
totuuden lähde: arkkitehtuuri, BLE-protokolla, UI-layout, osalista, avoimet kysymykset,
toteutusjärjestys.

## Sisältö
- `Tackcall-BLE-toistin-valmistusohje-vN.md` — päädokumentti (uusin N = ajantasainen)

## Periaate
Suunnittele-ennen-koodia: spesifikaatio kirjoitetaan ja hyväksytään ennen toteutusta.
Muutokset kirjataan versioituna (muutoshistoria dokumentin alussa). Avoimet kysymykset
pidetään auki kunnes kenttädata (vesitesti) ratkaisee ne — ei arvata pöydältä.
